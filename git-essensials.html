Getting Started

Open terminal at your project directory.

git init: Initialize a repository.(add .git file)

ls -la .git: Show inside .git file.

cat .git/config : You can check configuration. (URL,branch,remote)

git add <directory> : Hey, Git. Add all changes of that directory. After that you should commit.

git commit -m "message": commit the repository with message.The mesage gives infos about it.Your message in present tense."Fix for a bug" or "Fixes a bug" not "fixed a bug". Message answer this question: 'What is inside?'

  • Make changes
  • Add the changes
  • Comiit changes to the repository with a message.

git log : show logs about the project.

For example

    commit 809c9bb775581f971e63b3fb5bbbd2cd0fcce21f (HEAD -> testMd)
Author: fatihaysel <fatih.aysel@tosanalytics.com>
Date:   Sun Dec 13 21:16:50 2020 +0300

    First Commit

git log -n 5:show last 5 commits.

git log --since=2019-01-01: show commits after 2019.

git log --until=2019-01-01: show commits before 2019.

git log --author="name":Show commits who author is name. You don't need write fullname. You write just include strings from Author(Author:"fatihaysel"--git log --author="ihay").

git log -grep="Message":Show commits according to messages.(grep:gloabally search for regular expression.)

git log --until=2020-09-10 -n 2: show last two commits that happened before September 10, 2020.
